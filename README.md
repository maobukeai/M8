## 0. 快速开始

### 0.1 安装与启用

1. 将 `M8` 文件夹放入 Blender 插件目录（或通过“编辑 > 偏好设置 > 插件 > 安装”安装压缩包）。
2. 在“编辑 > 偏好设置 > 插件”中搜索 `M8` 并勾选启用。
3. 启用后你将获得：
   - 3D 视图右侧 N 面板：分类 `m8`  
     - **M8 常用工具**
     - **网格清理**
     - **尺寸专家工具 Pro**
     - **工具箱**
     - **智能烘焙重命名**
   - 多个快捷键增强（可在偏好设置中逐项开关）。
   - 顶部菜单栏（Topbar）追加“重启 Blender”按钮（可开关）。

> 推荐入门路线：先看 [快捷键总览](#02-默认快捷键总览最常用入口)，再按需求重点阅读 [尺寸专家工具 Pro](#5-模块三尺寸专家工具-pro)、[保存与导出](#15-模块十三保存与导出ctrls)、[模式切换](#9-模块七模式切换tab与对象设置)。

## 2. 插件偏好设置

位置：**编辑 > 偏好设置 > 插件 > M8**  

### 2.1 核心功能开关

可单独开启/关闭以下模块的快捷键注册：
- 变换辅助饼菜单（Shift+S）
- 模式切换（Tab）
- 智能工具（1/2/3/4）
- 成组工具（Ctrl+G）
- 快速删除（Object：X/Del）
- 删除饼菜单（Edit Mesh：X/Del）
- 对齐饼菜单（Alt+A）
- 着色饼菜单（Z）
- 保存饼菜单（Ctrl+S）
- 高级重命名（F2）
- 镜像（Shift+Alt+X）
- 顶栏“重启 Blender”按钮

### 2.2 快捷键冲突处理

遇到快捷键不生效时：
1. **置顶（Priority）**：提高 M8 键位优先级。
2. **独占（Exclusive）**：自动禁用其它插件的冲突项。
   - 推荐流程：先点“置顶”，不生效再点“独占”，想恢复点“恢复冲突”。

> 小提示：如果你已经在用其它插件占用了同一组按键，最稳妥的方案是——在 M8 偏好设置中关闭该模块的快捷键增强，只保留你需要的模块。

### 2.3 自动化功能

- **新建物体原点到底部**：新建 Mesh/Curve 等对象时自动将原点置于底部（可选自动落地）。
- **保存时自动打包/清理**：保存文件时自动打包资源并清理孤立数据。

## 3. 模块一：M8 常用工具（N 面板）

- **入口**：3D 视图 N 面板 > `m8` > **M8 常用工具**
- **适用**：物体模式（部分工具需要选中 Mesh）
- **用途**：把材质整理、合并、复制对齐等常用杂项集中到一个面板

**功能列表：**
- **材质工具**：
  - **材质排序**：按名称（A-Z/Z-A）或使用面数排序材质槽；可选移除未使用材质。
- **物体合并**：
  - **合并相近物体**：设定距离阈值，自动合并几何中心靠得很近的物体。
- **复制与对齐**：
  - **批量复制并对齐**：将激活物体（Source）复制并对齐到其他选中物体（Targets）的位置/旋转。
  - **目标处理**：可选择隐藏、删除、保留或移动目标物体到备份集合。
  - **原点对齐法向**：将物体原点旋转对齐到其所有面的平均法向（用于修复导入模型的轴向问题）。

## 5. 模块三：尺寸专家工具 Pro

- **入口**：3D 视图 N 面板 > `m8` > **尺寸专家工具 Pro**
- **适用**：物体模式（Object Mode）

**用途：**给一组网格生成“尺寸调节盒”，通过改盒子尺寸来精确缩放；并支持快照/还原与归档备用盒。

**核心功能：**
- **创建调节盒**：为选定对象生成尺寸控制盒，非破坏性缩放。
- **尺寸驱动**：修改盒子尺寸即可精确缩放内部物体。
- **快照/还原**：保存当前尺寸状态，随时一键还原（支持重建盒子或仅还原数据）。
- **备用盒**：归档当前盒子，稍后可随时恢复继续调节。
- **完成**：应用缩放（烘焙）或仅删除盒子（保留变换）。

**推荐流程：**
1. 物体模式选择一组网格对象。
2. 点击“创建调节盒”并选中盒子。
3. 直接修改盒子 Dimensions 或缩放盒子，实现整体精确缩放。
4. 需要设定基准时点击“保存快照(当前)”。
5. 完成时选择“完成(不烘焙/烘焙/保留盒)”之一。

## 7. 模块五：智能烘焙重命名（N 面板）

- **入口**：3D 视图 N 面板 > `m8` > **智能烘焙重命名**
- **适用**：物体模式（烘焙命名工作流）
- **用途**：为 Low/High 烘焙工作流提供快速命名、自动配对与批量整理

专为烘焙工作流（Substance Painter / Marmoset）设计的命名工具：
- **手动工具**：一键将选中物体设为 `_low` 或 `_high` 后缀。
- **智能重构（Smart Match）**：
  - 基于位置聚类（Graph Cluster）和面数判定。
  - 自动识别高低模对，并批量重命名为 `Name_001_low` / `Name_001_high`。
  - 支持自动放入 Collection。
- **传统匹配**：基于现有名称前缀匹配附近的对应物体。
- **查重叠**：检测场景中位置重叠的物体。

**推荐流程：**
1. 先用“手动工具”把一部分对象明确标为 `_low` / `_high`（可选）。
2. 使用“智能重构”自动配对并统一命名。
3. 如果你已经有成套命名，再用“传统匹配”做增量修复。
4. 出现错误配对时先用“查重叠”排查阈值与重叠对象。

## 9. 模块七：模式切换（Tab）与对象设置

- **快捷键**：<kbd>Tab</kbd>
- **适用**：所有模式（会随对象类型变化）
- **用途**：在“轻按切换模式”的基础上，扩展为“根据对象类型弹出快捷设置入口”
- **轻按**：切换 Object/Edit 模式（或进入对应对象特定模式）。
- **长按**：弹出“模式切换饼菜单”（含针对不同对象的快速设置）。

### 9.1 针对不同对象的菜单增强

当选中不同类型物体时，长按 Tab 会弹出不同内容：

- **网格（Mesh）**：
  - 切换点/边/面模式。
  - **优化菜单**：合并顶点（按距离）、有限融并、删除游离、重算法线、非流形检测。
  - **表面滑动**：快速调用顶点/边滑动。
  - **绘制所选**：在纹理绘制模式下绘制选中区域。
  - **常用开关**：透视/遮挡、自动合并顶点、法线朝向显示。

- **曲线/文字/曲面**：
  - **编辑工具**：手柄类型（Auto/Align/Vector）、细分、平滑、反转方向。
  - **快速设置**：弹出面板设置分辨率、挤出、倒角、填充模式。
  - **切换闭合**：一键切换曲线闭合状态。
  - **转为网格**：提供“保留原物体”或“直接替换”两种选项。

- **灯光（Light）**：
  - 快速调整能量、半径/角度。
  - **对准选中**：让灯光自动指向当前选中的物体。

- **摄像机（Camera）**：
  - 快速调整焦距、曝光。

## 20. FAQ

### 20.1 快捷键冲突怎么办？
请前往插件偏好设置，找到对应模块，先点“置顶”，若无效再点“独占”。

### 20.2 智能工具（1/2/3/4）覆盖了我的层级显示？
这是 Blender 默认的 Collection 显隐快捷键。如果你更习惯用数字键控制层级，请在 M8 偏好设置中关闭“智能工具”模块。

### 20.3 尺寸专家工具还原不准？
如果你执行过“完成(烘焙)”，物体的缩放已被应用为 1，此时无法仅通过数据还原，必须依赖快照。若物体含有 Shape Keys，数据还原可能会被限制以保护形态键。
